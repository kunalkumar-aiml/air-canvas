# Air Canvas

A real-time air drawing application built using hand tracking.

## Overview

This project allows users to draw in the air using their index finger. 
The application tracks hand landmarks through the webcam and renders 
the drawing on a virtual canvas in real time.

## Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy

## Setup

pip install -r requirements.txt

## Features

- Real-time hand tracking
- Air drawing with index finger
- Clear canvas option
- Save drawing locally

## Controls

C - Clear canvas  
S - Save drawing  
Q - Quit

## Run

python main.py

Press 'q' to exit.
